# Flashcards

A flashcard app for iOS built with SwiftUI and SwiftData.

## Features

- Create and manage decks of flashcards
- Add cards with a front (question) and back (answer)
- Review session with weighted queue — cards with lower accuracy appear more frequently
- Per-deck stats showing total cards and mastery percentage
- Sort decks by name or mastery rating

## Running the project

Clone the repo and open `Flashcards.xcodeproj` in Xcode
